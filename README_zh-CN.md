# ScreenMark - ITCG 屏幕水印工具

> 中文 | [English](README.md)

**ScreenMark** 是一款专业的 Windows 屏幕水印工具，旨在通过未经授权的拍照和截图来防止信息泄露。它在屏幕上直接叠加可自定义的水印，为数据安全提供强大的视觉威慑。

**版本：** 3.3.7  
**平台：** Windows 7+ (.NET 8.0)  
**授权：** 免费（可选专业版升级）

---

## 功能特性

### 四种水印类型

| 类型 | 说明 | 可用版本 |
|------|------|---------|
| **文字水印** | 自定义文本、动态变量、字体、颜色 | 免费版 |
| **图片水印** | PNG/JPG/BMP 图片叠加显示 | 专业版 |
| **3D模型水印** | GLB/GLTF/OBJ/STL/3DS 3D模型展示 | 专业版 |
| **二维码水印** | 动态生成含用户信息的QR码 | 专业版 |

### 动态变量

在水印文本中插入实时动态变量，每秒自动刷新：

| 变量 | 说明 |
|------|------|
| `{username}` | 当前 Windows 登录用户名 |
| `{ip}` | 本机 IPv4 地址 |
| `{hostname}` | 计算机名称 |
| `{mac}` | 网卡 MAC 物理地址 |
| `{date}` | 当前日期（年-月-日） |
| `{time}` | 当前时间（时:分:秒） |
| `{newline}` | 换行符 |

### 动画效果

- **水平移动** — 水印在屏幕上左右弹跳移动
- **垂直移动** — 水印在屏幕上上下弹跳移动
- **对角移动** — 同时启用水平和垂直方向
- **速度可调** — 1-100 级精确控制
- **阴影特效** — 可调节阴影强度

### 水印平铺

- 全屏重复显示水印，有效防止拍照和截图泄露
- 可调节水平和垂直间距控制水印密度
- 文字以 45 度角倾斜显示，增强防伪效果
- 使用 WPF `VisualBrush` 实现高性能渲染

### 桌面背景覆盖

- **图片背景** — 设置自定义图片作为全屏背景覆盖
- **颜色背景** — 纯色覆盖，支持渐变效果
- **渐变模式** — 水平、垂直、对角线三种渐变方向
- 按 **ESC 键** 取消背景覆盖

### 3D 模型水印（专业版）

- 支持 **GLB、GLTF、OBJ、STL、3DS** 格式的 3D 模型导入
- 可调节 **缩放比例**（0.1-5倍）、**旋转速度**（0-180°/秒）、**旋转轴**（X/Y/Z）
- 各轴角度独立调节（0-360°）
- **窗口大小** 可配置或 **铺满全屏** 模式
- **三种光影模式：**
  - **简单** — 基础环境光 + 方向光
  - **标准** — 三点照明（主光、补光、边缘光）
  - **电影级** — 多光源冷暖色调对比
- **光照控制：** 镜面反射强度、环境光强度、主光源颜色/强度
- **边缘光** 和 **动态旋转光照** 特效
- 内置默认 3D 模型

### 多屏幕支持

- 自动检测所有连接的显示器
- 指定水印显示在特定屏幕上
- 每个屏幕独立的位置和大小配置
- 随时刷新屏幕列表

### 预设管理

- 将完整的水印配置保存为命名预设（JSON 格式）
- **内置 5 个预设模板：**
  - 企业防泄漏
  - 个人标识
  - 日期时间标记
  - 全屏平铺防拍
  - 二维码溯源
- 从设置界面或系统托盘快速切换预设
- 预设文件存储在 `Presets/` 目录，方便备份和分享

### 安全口令

- 设置安全口令保护水印配置
- 打开设置面板或退出程序时需要输入密码
- 防止他人未经授权修改水印配置或关闭水印保护

### 企业 / 无盘模式

专为企业环境设计，支持集中 IT 管理：

- **设备专属水印** — 按 IP 地址为每台设备分配独立水印
- **公共水印** — 所有设备共用的统一水印
- **管理员 IP 限制** — 仅授权 IP 地址可以修改设置
- **批量授权** — 多设备许可证管理
- **配置文件** — `EnterpriseConfiguration.ini` 基于 IP 的规则管理

### 屏幕聚焦模式（隐私遮罩）

- 按 `Ctrl + Shift + F` 开启/关闭聚焦模式
- 当前活跃窗口保持明亮 + **发光边框高亮**，其余屏幕区域自动变暗
- 鼠标穿透设计 — 点击暗色区域可正常操作下方窗口，无需退出
- 切换窗口或拖拽窗口时高亮区域**实时跟随**，零延迟响应
- 在系统设置中可调节遮罩不透明度（20%-95%，默认 70%）
- 按 `ESC` 或再次按下 `Ctrl + Shift + F` 退出
- 支持多显示器环境
- **完全免费**，无需注册或授权

### USB 设备插入告警

- 基于 WMI 底层事件实时检测 U 盘、移动硬盘的插入/移除
- 未授权设备插入时，系统托盘弹出 **气泡告警通知**，同时写入日志文件
- 捕获完整的设备信息：盘符、卷标、文件系统、容量、硬件序列号、插入时间
- **设备白名单机制** — 公司配发的 U 盘加入白名单后不会触发告警，支持按序列号或盘符匹配
- 一键添加当前已插入设备到白名单，支持手动编辑和批量管理
- 有效防止内部数据通过 USB 存储设备被非法拷贝
- **完全免费**，无需注册或授权

### 多语言支持

- **中文（zh-CN）** 和 **英文（en-US）** 完整界面
- 首次运行自动检测系统语言
- 在系统设置「语言设置」中可随时手动切换
- 设置面板全部标签页、按钮、滑块、下拉选项、弹窗消息、系统托盘菜单均已翻译

## 安装说明

### 环境要求
- Windows 7 或更高版本
- .NET 8.0 运行时

### 下载
从 [https://sm.itcg.cn/](https://sm.itcg.cn/) 下载最新版本。

### 快速开始
1. 解压 ZIP 压缩包
2. 运行 `ScreenMark.exe`
3. 右键点击系统托盘图标，选择「打开设置」
4. 配置水印文本和参数
5. 点击「保存并关闭」应用设置
